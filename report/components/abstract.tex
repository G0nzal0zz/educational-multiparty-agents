\begin{abstract}
\noindent
The integration of conversational agents in educational environments has opened new frontiers in the landscape of education. While most modern approaches explore dyadic settings where an agent instructs a human learner, we implement a conversational system based on a triadic classroom comprising an agentic teacher, an agentic student, and a human learner. By allowing human learners to overhear and participate in agentic interaction, we aim to reduce cognitive load and improve knowledge retention. Evaluation with human participants suggests that the presence of an autonomous student agent enhances engagement and simulates a dynamic classroom experience, resulting in similar cognitive test performance but significant improvements in the learning experience of the human.

\end{abstract}
