\begin{abstract}
\noindent
    This project explores vicarious learning within a real-time spoken dialog framework. 
    We present a multi-agent system where a Teacher agent delivers educational content while a Student agent acts as a synthetic peer, 
    interjecting with clarifying questions to draw out more detailed explanations. Unlike traditional text-based tutors, 
    our system utilizes a low-latency speech interface, integrating Whisper for STT and Chatterbox for TTS, to foster a natural, 
    triadic "fictive dialog" environment. By allowing human learners to overhear and join this agentic interaction, 
    we aim to reduce cognitive load and improve knowledge retention. 
    Evaluation with human participants suggests that the presence of an autonomous student agent enhances engagement and simulates a dynamic classroom experience.


\end{abstract}
